SQLite tables the MCP server reads:
```sql
-- repos: id, path, name, type, last_commit, scanned_at
-- services: id, repo_id, name, root_path, language
-- connections: id, source_service_id, target_service_id, protocol, method, path, source_file, target_file
-- schemas: id, connection_id, role, name, file
-- fields: id, schema_id, name, type, required
-- connections_fts: FTS5 virtual table on (path, protocol, source_file, target_file)
```

Recursive CTE for transitive impact (from design doc):
```sql
WITH RECURSIVE impacted(id, depth, path) AS (
  SELECT target_service_id, 1, source_service_id || ',' || target_service_id
  FROM connections WHERE source_service_id = ?
  UNION ALL
  SELECT c.target_service_id, i.depth + 1, i.path || ',' || c.target_service_id
  FROM connections c JOIN impacted i ON c.source_service_id = i.id
  WHERE i.path NOT LIKE '%,' || c.target_service_id || ',%'
    AND i.depth < 10
)
SELECT DISTINCT id FROM impacted;
```
