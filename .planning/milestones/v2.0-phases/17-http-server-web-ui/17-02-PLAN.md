  <action>
Create `worker/ui/force-worker.js` as a Web Worker script that runs the D3 force simulation off the main thread.

The worker:
1. Imports D3 force modules via ESM CDN:
   ```javascript
   import { forceSimulation, forceLink, forceManyBody, forceCenter, forceCollide } from 'https://cdn.jsdelivr.net/npm/d3-force@3/+esm';
   ```

2. Listens for messages from the main thread. Message types:
   - `{ type: 'init', nodes, links, width, height }` — initializes simulation
   - `{ type: 'reheat' }` — calls `simulation.alpha(0.3).restart()`
   - `{ type: 'stop' }` — calls `simulation.stop()`
   - `{ type: 'drag', nodeId, x, y }` — fixes a node position during drag

3. On `init`: creates simulation with forces:
   ```javascript
   const simulation = forceSimulation(nodes)
     .force('link', forceLink(links).id(d => d.id).distance(100).strength(0.8))
     .force('charge', forceManyBody().strength(-300))
     .force('center', forceCenter(width / 2, height / 2))
     .force('collide', forceCollide(22));
   ```

4. On each simulation tick, posts `{ type: 'tick', nodes }` back to the main thread with current node positions (x, y for each node). Uses `simulation.on('tick', ...)`.

5. On simulation end (`simulation.on('end', ...)`), posts `{ type: 'end' }`.

6. The simulation runs 300 tick iterations then stops (`alphaDecay` tuned for quick stabilization or explicit `stop()` after 300 ticks).

Important: The worker receives node/link arrays. Node objects from the main thread are plain `{id, name, ...}`. D3 force simulation mutates these with x, y, vx, vy properties. Post back only the position data needed for rendering: `{ id, x, y }` per node.

**Initialization:**
```javascript
async function init() {
  const canvas = document.getElementById('graph-canvas');
  const container = document.getElementById('canvas-container');

  // Resize canvas to fill container
  function resize() {
    canvas.width = container.clientWidth;
    canvas.height = container.clientHeight;
    render();
  }
  window.addEventListener('resize', resize);
  resize();

  // Load graph data
  const resp = await fetch('/graph');
  if (!resp.ok) {
    document.getElementById('node-info').textContent = 'No map data yet. Run /allclear:map first.';
    return;
  }
  graphData = await resp.json();
  document.getElementById('node-info').textContent = `${graphData.nodes.length} services, ${graphData.edges.length} connections`;

  // Initialize positions to random before simulation
  graphData.nodes.forEach(n => {
    positions[n.id] = { x: Math.random() * canvas.width, y: Math.random() * canvas.height };
  });

  // Start Web Worker force simulation
  forceWorker = new Worker('./force-worker.js', { type: 'module' });
  forceWorker.onmessage = handleWorkerMessage;
  forceWorker.postMessage({
    type: 'init',
    nodes: graphData.nodes.map(n => ({ id: n.id, ...positions[n.id] })),
    links: graphData.edges.map(e => ({ source: e.source_service_id, target: e.target_service_id })),
    width: canvas.width,
    height: canvas.height,
  });

  setupInteractions(canvas);
  setupControls();
}
```

**Worker message handler:**
```javascript
function handleWorkerMessage({ data }) {
  if (data.type === 'tick') {
    data.nodes.forEach(({ id, x, y }) => { positions[id] = { x, y }; });
    render();
  }
}
```

**Render function** (Canvas, not SVG):
- Apply `transform` (pan/zoom) via `ctx.save(); ctx.translate(tx); ctx.scale(s); ctx.restore()`
- Determine visible nodes: nodes whose name includes `searchFilter` (case-insensitive)
- Determine selected/blast sets: if `selectedNodeId` set, find direct neighbor IDs; if `blastNodeId` set, fetch from `/impact?change=<name>` (cache result)
- Draw edges first, then nodes on top:
  - Each edge: draw a line from `positions[source_service_id]` to `positions[target_service_id]`
  - Skip edges where `!activeProtocols.has(edge.protocol)`
  - Skip edges where both endpoints are not in the visible (search-filtered) set
  - Color: `COLORS.edge.selected` if edge connects to selectedNodeId, `COLORS.edge.blast` if in blast path, else `COLORS.edge.default` if no selection active, else `COLORS.edge.dimmed`
  - Each node: draw a circle at `positions[n.id]` with radius NODE_RADIUS
  - Color: `COLORS.node.selected` if selectedNodeId, `COLORS.node.blast` if in blastSet, `COLORS.node.default` if no selection or in visible filtered set, `COLORS.node.dimmed` otherwise
  - Draw label below the circle: service name, capped at 12 chars + ellipsis if longer

**Interactions:**
- `mousemove`: iterate nodes, compute `Math.hypot(ex - pos.x, ey - pos.y)` for each, if < NODE_RADIUS show cursor pointer + tooltip; if dragging, send `{ type: 'drag', nodeId, x, y }` to worker
- `mousedown`: hit test for node; if found set `selectedNodeId`, set `isDragging=true, dragNodeId`, reheat simulation
- `mouseup`: stop drag, fix node position (set `node.fx = node.x, node.fy = node.y`)
- `click` (no drag): if node hit, set `selectedNodeId = node.id`; if same node clicked twice, clear selection; if Shift+click, set `blastNodeId = node.id` and fetch impact
- Canvas `wheel`: zoom — update `transform.scale` clamped between 0.2 and 5, center on cursor position

**Search control:**

```javascript
document.getElementById('search').addEventListener('input', e => {

  render();

```

**Protocol filters:**
```javascript
document.querySelectorAll('[data-protocol]').forEach(cb => {
  cb.addEventListener('change', () => {

    else activeProtocols.delete(cb.dataset.protocol);
```
