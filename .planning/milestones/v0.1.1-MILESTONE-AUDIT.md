## Integration Check

Cross-phase wiring audited via gsd-integration-checker. **1 gap found and
fixed before close:**

### Gap 1 — FIXED

**Issue:** `plugins/arcanon/scripts/session-start.sh:159` read stale JSON
key `hub_auto_upload` after Phase 97 renamed the emitter key in
`worker/cli/hub.js` to `hub_auto_sync`. Users with `auto_sync` enabled
saw "Hub: manual" in the banner instead of "Hub: auto-sync on".

**Fix:** Commit `802d59b` — change jq filter to
`.hub_auto_sync // .hub_auto_upload // false` (reads new key with legacy
fallback).

**Affected REQs:** SSE-01, SSE-02. Now passing.
