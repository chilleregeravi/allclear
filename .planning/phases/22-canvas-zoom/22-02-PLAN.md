Key browser convention (from research):
- e.ctrlKey === true  → pinch-to-zoom gesture (trackpad) OR Ctrl+scroll (mouse) → ZOOM
- e.ctrlKey === false → two-finger scroll (trackpad) OR plain mouse wheel → PAN for trackpad, ZOOM for mouse
  Note: plain mouse wheel also has ctrlKey===false, but deltaX===0, so the pan-on-scroll behavior
  is correct for mouse too (panning by 0 on X, zooming via deltaY with ctrlKey check misses mouse scroll).

IMPORTANT: The research recommends a ctrlKey split where ctrlKey=false becomes pan. However, a plain
mouse wheel also sends ctrlKey=false. The correct behavior for mouse wheel is still zoom. The distinction:
trackpad two-finger scroll sends non-zero deltaX OR small deltaY values; plain mouse wheel sends large
deltaY with deltaX=0. To support both correctly: treat ctrlKey=true as zoom always, and ctrlKey=false
as pan (moving state.transform by deltaX/deltaY). Mouse wheel users on ctrlKey=false will pan, not zoom —
this is acceptable (mouse users can Ctrl+scroll to zoom, which is the standard shortcut). Trackpad users
get the natural gesture mapping.

```javascript
// { passive: false } is required — without it, e.preventDefault() is silently ignored
// and the browser scrolls the page instead of zooming the canvas.
canvas.addEventListener(
  "wheel",
  (e) => {
    e.preventDefault();

    if (e.ctrlKey) {
      // Pinch-to-zoom (trackpad) or Ctrl+scroll (mouse) — ZOOM
      // D3-style continuous delta: normalize across deltaMode, apply sensitivity factor
      const SENSITIVITY = 0.001; // lower = slower zoom; D3 default is 0.002
      const rawDelta = -e.deltaY * (e.deltaMode === 1 ? 0.05 : e.deltaMode ? 1 : SENSITIVITY);
      const factor = Math.pow(2, rawDelta); // exponential feels natural vs linear
      const newScale = Math.min(5, Math.max(0.15, state.transform.scale * factor));
      const ratio = newScale / state.transform.scale;
      state.transform.x = e.offsetX - ratio * (e.offsetX - state.transform.x);
      state.transform.y = e.offsetY - ratio * (e.offsetY - state.transform.y);
      state.transform.scale = newScale;
    } else {
      // Two-finger scroll (trackpad) or plain mouse wheel — PAN
      state.transform.x -= e.deltaX;
      state.transform.y -= e.deltaY;
    }

    render();
  },
  { passive: false },
);
```
