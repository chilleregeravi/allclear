<task type="auto">
  <name>Task 1: Create hook manifest and lint script</name>
  <files>hooks/lint.json, scripts/lint.sh</files>
  <action>
Create `hooks/lint.json` — a Claude Code hook manifest that registers `scripts/lint.sh` for PostToolUse events on Edit and Write tools. Follow the same manifest format as other Claude Code plugin hooks (array of hook objects with `hook_event_name`, `tool_name` matcher, and `hook_file_path` referencing `${CLAUDE_PLUGIN_ROOT}/scripts/lint.sh`). Register TWO entries: one for tool_name "Edit" and one for tool_name "Write".

Create `scripts/lint.sh` — a bash script with `#!/usr/bin/env bash` and `set -euo pipefail`. The script must:

**Header guards (in order):**
1. Check `ALLCLEAR_DISABLE_LINT` env var — if set, `exit 0` immediately
2. Route ALL debug output to stderr: `exec 2>/dev/null` (PLGN-08)
3. Read stdin JSON via `INPUT=$(cat)` and extract `tool_input.file_path` and `tool_name` using jq (PLGN-07 pattern: `printf '%s\n' "$INPUT" | jq -r '.tool_input.file_path // empty'`)
4. If file path is empty or file doesn't exist, `exit 0`
5. Skip files in generated directories: if path contains `node_modules/`, `.venv/`, `target/`, `dist/`, `build/`, or `__pycache__/` — `exit 0`

**Language detection:**
Source `${CLAUDE_PLUGIN_ROOT}/lib/detect.sh` and call `detect_language "$FILE"` to get the language. If `lib/detect.sh` is not available (Phase 2 not yet executed), fall back to extension-based detection inline:
- `.py` → python
- `.rs` → rust
- `.ts`, `.tsx`, `.js`, `.jsx` → typescript
- `.go` → go
- Anything else → `exit 0` (no linter for this file type)

**Linter invocation by language:**

Python (LNTH-02):
```bash
if command -v ruff &>/dev/null; then
  LINT_OUTPUT=$(ruff check "$FILE" 2>&1 || true)
  LINTER_NAME="ruff"
fi
```

Rust (LNTH-03) — with 30-second throttle:
- Resolve Cargo.toml root via `cargo locate-project --message-format plain 2>/dev/null | xargs dirname 2>/dev/null || true`
- If no Cargo.toml root found, exit 0
- Compute throttle key from Cargo root path: `printf '%s' "$CARGO_ROOT" | cksum | cut -d' ' -f1` (cksum is POSIX, unlike md5sum which may be absent on macOS)
- Throttle file: `/tmp/allclear_clippy_${THROTTLE_KEY}`
- Read `ALLCLEAR_LINT_THROTTLE` env var (default 30 seconds)
- Compare `$(date +%s)` to stored timestamp — skip if fewer than THROTTLE_SECS elapsed
- If not throttled AND `command -v cargo` exists: `cd "$CARGO_ROOT" && cargo clippy --message-format=short 2>&1 || true`
- Write current timestamp to throttle file after starting clippy
- Set `LINTER_NAME="clippy"`

TypeScript/JavaScript (LNTH-04):
- Resolve eslint binary with fallback chain: `node_modules/.bin/eslint` (check existence) → `$(npm bin 2>/dev/null)/eslint` → `command -v eslint`
- If no eslint binary found, exit 0 silently (LNTH-08)
- Run: `LINT_OUTPUT=$("$ESLINT" "$FILE" 2>&1 || true)`
- Check exit status of eslint: if the captured output contains "Oops!" or "ESLint couldn't find" (config error), treat as skip — clear LINT_OUTPUT
- Set `LINTER_NAME="eslint"`

Go (LNTH-05):
- If `command -v golangci-lint` exists:
  - Resolve package directory: `PKG_DIR=$(dirname "$FILE")`
  - Run: `LINT_OUTPUT=$(golangci-lint run "${PKG_DIR}/..." 2>&1 || true)`
  - Set `LINTER_NAME="golangci-lint"`

**Output formatting (LNTH-06):**
- If LINT_OUTPUT is empty or whitespace-only, emit nothing and `exit 0`
- Truncate to first 30 lines; if more exist, append `"... (N more lines — run \`<linter> <file>\` to see all)"`
- Format message: `AllClear lint [$LINTER_NAME]: $FILE\n$TRIMMED_OUTPUT`
- Emit JSON: `printf '{"systemMessage": %s}\n' "$(printf '%s' "$MSG" | jq -Rs .)"`

**Final exit:** Always `exit 0` — this is non-negotiable (LNTH-07).

Make `scripts/lint.sh` executable: `chmod +x scripts/lint.sh`.
